# scala-cli-millbuild-sieve-atkins

## Description
Given a number find all primes.

## Tech stack
- scala
- millbuild

## Docker stack
- nightscape/scala-mill

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
